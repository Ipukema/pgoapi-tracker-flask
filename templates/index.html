pgoapi-tracker

welcome to the home of the pgoapi-tracker.

this site is currently a work in progress, but the following features will be implemented:

  • oauth2 logins for google and for users of the pgoapi slack channel
  • standard login using ptc
  • ability to track locations simulated via. the pgoapi
    • this will use google maps as the maps engine
    • i plan to set up custom urls for each user to post their locations to
    • eventually, i will create a python module for use with pgoapi to send the location information
  • view caught pokemon
  • list inventory items
  • see nearby pokemon, pokestops and gyms
  • incubate eggs
  • hatch eggs
  • evolve pokemon
